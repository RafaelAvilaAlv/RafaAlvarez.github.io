Sobre mí

Soy Tecnólogo Superior Universitario en Desarrollo de Software, con experiencia en el desarrollo de aplicaciones web escalables y APIs REST seguras.

Trabajo principalmente con Java, Spring Boot y Angular, aplicando buenas prácticas de arquitectura, seguridad con JWT y despliegue en la nube.

Me caracterizo por la mejora continua, el trabajo en equipo y la entrega de soluciones con impacto real en el usuario.

Predicción Inteligente de Ranking Universitario

Sistema inteligente que predice el ranking QS 2025 utilizando Spring Boot, regresión, arquitectura limpia, JWT, Swagger y Docker.

Sistema de Gestión de Residencias

Plataforma web desarrollada con Angular y Spring Boot para la gestión de residencias estudiantiles con roles y persistencia en MySQL.

Chatbot con API OpenAI

Chatbot backend desarrollado en Spring Boot conectado a la API de OpenAI, con manejo de sesiones y conversaciones dinámicas.

CRUD Web en Azure

Aplicación CRUD desplegada en Azure App Service, integrando frontend y backend para pruebas en la nube.
